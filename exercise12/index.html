I used the basic template with a spring palette for my base layer. I changed the colors of the water and greenspace to match the colors in the base palette because I liked the green and blue from that palette but with the spring base color. I also changed the color of roadways to match the color that the streets template uses. I also changed the font color of places to be darker depending on if it was major or minor as well as adjusting the zoom for the place labels. The last change I made was changing and reducing the line opacity for roads so that it wouldn't be as distracting. I also got rid of the halo around the street labeling to make it easier to read.


I tried adding outside shapefiles to add to the map but I don't think it was within the right parameters so I decided to make a small dataset to see if it would work. I added a couple bars that I like in Des Moines and a polygon shape around West Glen in my additional layer. It shows in the layers page but doesn't show up on the actual map.
